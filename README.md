# Python File Organizer

A Python automation script that organizes files into folders based on their extensions.

## Features

- Creates folders automatically
- Sorts images
- Sorts videos
- Sorts documents
- Sorts script files
- Ignores existing directories

## Technologies Used

- Python
- os module
- shutil module

## Supported Extensions

Documents:
- .pdf
- .docx
- .pptx
- .xlsx

Images:
- .png
- .jpg
- .jpeg
- .gif

Videos:
- .mp4
- .mkv
- .avi

Scripts:
- .py
- .js
- .html
- .css
